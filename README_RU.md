# BOONWAVE v5.5.0

Готовая PWA-сборка для публикации на Netlify.

## Установка
1. Загрузите содержимое этой папки на Netlify так, чтобы `index.html` находился в корне.
2. Откройте ссылку в Safari на iPhone.
3. Нажмите «Поделиться» → «На экран Домой».

## В сборке
- утверждённый логотип BOONWAVE и отдельный знак;
- экран загрузки и вход/регистрация;
- пространства «Личное» и «Проекты»;
- проекты, цели, люди, идеи и этапы;
- перемещение и масштабирование рабочего поля;
- локальное хранение и офлайн-кэш;
- принудительное обновление service worker;
- PWA-иконки 192×192 и 512×512.

Данные сохраняются локально в браузере устройства. Облачная синхронизация не подключена.
