# BOONWAVE v6.0.48 — Я СЕЙЧАС

Тестовая ветка режима оперативного фокуса.

## Изменения
- кнопка S заменена фирменной линейной иконкой человека с поднятыми руками;
- режим называется «Я сейчас»;
- исправлен слой панели: список отображается поверх размытого фона на iPhone Safari;
- задачи собираются из всех рабочих процессов текущего пространства;
- сортировка: просроченные → сегодня → ближайшие 48 часов → остальные, внутри — по приоритету и сроку;
- задачу можно отметить выполненной прямо в панели;
- отметка синхронно обновляет исходный рабочий процесс, прогресс и счётчики;
- доступна отмена последней отметки.

Версия остаётся тестовой до проверки на реальном iPhone.
