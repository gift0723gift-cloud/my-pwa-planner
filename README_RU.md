# BOONWAVE v7.0.0 — CLEAN ARCHITECTURE

Это новая чистая ветка, собранная без наследования конфликтующих CSS-патчей и обработчиков v6.x.

Архитектура:
- `state.js` — единое состояние и контракты сущностей;
- `storage.js` — локальные профили, миграция v6, импорт/экспорт;
- `media.js` — IndexedDB для изображений и файлов;
- `render.js` — чистый рендер интерфейса;
- `canvas.js` — конечный автомат жестов холста и карточек;
- `links.js` — независимый слой связей;
- `main.js` — маршрутизация и координация.

Публикация: загрузить содержимое папки в корень GitHub Pages. Для установленной старой PWA удалить старый ярлык и установить заново после проверки Safari.
