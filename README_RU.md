# BOONWAVE v6.0.60 — CARD & GESTURE RECOVERY CLEAN

Исправления:

- восстановлена исходная логика кнопки «глаз»: одно нажатие переключает компактный/средний режим, двойное открывает подробную карточку;
- кнопка связи снова использует проверенный `startLinkCreation()`, а не отдельный неполный сценарий;
- глобальная панель привязана к выбранной карточке и пересчитывается после перемещения/изменения режима;
- исправлено преждевременное завершение pinch: карточка перестраивается только после отпускания обоих пальцев;
- двойной тап определяется на втором `pointerdown`, блокирует нативное Safari-масштабирование и одинаков для всех типов, включая «Человек»;
- добавлена блокировка передачи карточных жестов холсту.

Важно: поведение iOS требует проверки на реальном iPhone.
