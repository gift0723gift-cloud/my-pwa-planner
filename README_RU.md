- двойное нажатие открывает просмотр;
- удержание 2 секунды открывает редактирование;
- отдельная связанная карточка «Рабочий процесс»;
- этапы, задачи, люди и затраты перенесены из основной карточки;
- основные изображения, PDF и файлы остаются в проекте;
- в просмотре проекта добавлена горизонтальная микро-галерея;
- компактные уровни карточек и очищенное управление холстом.
